配置中心

配置行情下拉列表中的品种。展示名称可自定义；TradingView 代码需与 TradingView 中一致（如 BINANCE:BTCUSDT）。BINANCE: 前缀为加密（WS 订阅 K 线）；其余默认走长桥，若无法自动映射美股/港股代码，可在 JSON 中为该条增加 longPortSymbol（如 QQQ.US）。interval 与图表周期、长桥订阅周期、Binance K 线周期一致。应用只使用一份配置文件 config.json（路径见下），首次启动会从安装目录旁模板生成；保存配置即写回该文件。

LLM（OpenAI 兼容接口）

API Key 保存在本机用户目录配置文件；未填写时使用环境变量 OPENAI_API_KEY（配置优先）。启用分析需 ARGUS_ENABLE_LLM=1。

系统提示词

币圈与股票分开配置。分流规则与行情一致：BINANCE: 前缀或品种行 feed: crypto 为币圈；否则为股票/长桥。股票侧默认强调美股常规时段（美东 9:30–16:00）与盘前盘后差异。
